= ASA R2存储系统的ONTAP软件支持和限制
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
虽然ASA R2系统为SAN解决方案提供了广泛的支持、但某些ONTAP软件功能不受支持。

.ASA R2系统不支持以下功能：
* iSCSI LIF故障转移
* FabricPool
* LUN厚配置
* MetroCluster
* 对象协议
* ONTAP S3 SnapMirror和S3 API
* SnapMirror (异步和同步)到云
* SnapMirror (异步和同步)到非ASA R2系统
* 选择性 LUN 映射（ SLM ）


.ASA R2系统支持以下功能：
* Snaplock
* 双层加密
